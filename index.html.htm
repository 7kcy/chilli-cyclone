Setup guide

How to install

Option A — Single file
Download one pattern
01
Download the .ahk file
Click the download button on any field card above. The file is saved directly from GitHub.
02
Open Natro Macro
Launch Natro Macro as you normally would. Make sure you're running AutoHotkey v2.
03
Go to Patterns
In Natro, navigate to the Patterns section and open the patterns folder.
04
Drop in the file
Copy paste the downloaded .ahk file into the patterns folder. It will appear in the dropdown immediately.
05
Copy the settings
Hit Copy Settings on the field card, then paste into Natro's field config. Done.
Option B — Full pack
Download the zip
01
Download the zip
Hit Download zip in the release section below. This contains all available patterns in one file.
02
Extract the archive
Right-click the zip and choose Extract All. You'll get a folder with all the .ahk files.
03
Open Natro Macro
Launch Natro and navigate to the Patterns section, then open the patterns folder.
04
Move all files in
Copy every .ahk file from the extracted folder into Natro's patterns folder.
05
Select and configure
All patterns now appear in the dropdown. Use Copy Settings on each card to get the correct Natro config.
